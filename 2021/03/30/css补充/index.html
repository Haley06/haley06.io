flex布局

  • flex布局:flexible布局,弹性布局
  • 移动端最多,目前PC端也越来越多

1.概念

  • 开启了flex布局的元素叫flex container
  • flex container里面的子元素叫做flex item
  • 设为 Flex 布局以后，子元素的float、clear和vertical-align属性将失效。

开启flex布局两种方式

2.flex布局模型

  • flex container 中的两个轴:

    • 主轴:main axis(相当于x轴)

      主轴开始地方:main srtart

      ​ 结束地方:main end

      ​ 主轴大小:main size(相当于盒子宽)

    • 交叉轴:cross axis(相当于y轴)

      交叉轴开始地方:cross srtart

      ​ 结束地方:cross end
      ​ 交叉轴大学:cross size(相当于盒子高)

    • 默认情况在左上角为开始

3.与flex有关相关属性

用作container上:

  • flex-flow
  • flex-direction
  • flex-wrap
  • justify-content
  • align-items
  • align-content

用在flex items上的css属性

  • flex
  • flex-grow
  • flex-basis
  • flex-shrink
  • order
  • align-self

二 .给flex-items的属性

1.order

  • 可以设置任意整数(正整数,负整数,0),值越小越排在前边
  • 默认值是0

2.align-self

  • 这个可以修改align-items设置，专门应用于自己的item 的对齐方式,与其他对齐方式不同

3.flex-grow

flex-grow决定了flex items如何扩展

  • 可以设置任何非负数字(正小数,负小数,0),,默认值为0
  • 当flex container在main axis方向上有剩余时,flex-grow属性才会有效

如果所有flex-items的flex-grow总和sum超过1.每个item扩展大小为:

  • 剩余部分大小*flex-grow/sum

如果所有flex-items的flex-grow总和sum不超过1.每个item扩展大小为:

  • 剩余部分*flex-grow为长大 的大小

    flex items扩展后不能超过max-width\max-height

4.flex-shrink

==决定flex-item如何让收缩==

  • 可以设置任何非负数字(正小数,负小数,0),,默认值为0

  • 当flex container在main axis方向上有超出部分时,flex-shrink属性才会有效

  • 如果所有flex-items的flex-shrink总和sum超过1

    .每个item收缩大小为:

    =超出部分的size * 收缩比例 / 所有flex items收缩比例之和

  • 如果所有flex-items的flex-grow总和sum不超过1.

    每个item收缩大小=超出部分的size * sum * 收缩比例 / 所有flex items收缩比例之和

    收缩比例=flex-shrink * flex item的basesize

    basesize为item的之前的size在放入container里时

flex items扩展后不能小于max-width\max-height

5.flex-basis和flex

  • flex-basis决定主轴上item的大小，优先级较高

    ​ 分配多余空间之前,项目占据的主轴空间,默认为auto

  • flex属性:

    ​ 是flex-grow/shrink/basis的缩写,可以指定1个,2个或3个值

    单值语法: 值必须为其下之一:

    • 一个无单位数,会被当做flex-grow值
    • 一个有效的宽度值,会被当作flex-basis的值
    • 关键字:none,auto和initial

    双值语法: 第一个值必须无单位,并且会被当做flex-grow值

    • 第二个值:

      ​ 一个无单位数,flex-shrink

      ​ 一个有效宽度:flex-basis

    三值语法:无单位(flex-grow) 无单位(flex-shrink) 有效宽度(flex-basis)

区别

***align-items和align-content的区别**

  • align-items属性是针对单独的每一个flex子项起作用，它的基本单位是每一个子项，在所有情况下都有效果（当然要看具体的属性值）。

  • align-content属性是将flex子项作为一个整体起作用，它的基本单位是子项构成的行，只在两种情况下有效果：①子项多行且flex容器高度固定 ②子项单行，flex容器高度固定且设置了flex-wrap:wrap;

  • 注：这里的高度固定的意思实际上是让flex容器在交叉轴上有多余的空间。*

justify-content，justify-items和justify-self之间的主要区别：

  • justify-conten: 它不适用于或控制网格项目的对齐。

    displsy:flex时一般用justify-content属性

  • justify-items属性控制网格项目的对齐方式。它设置在网格容器上。

  • justify-self属性将覆盖单个项目上的justify-items。默认情况下，它在网格项目上设置并继承justify-items的值。

过度(Transition)

  • 什么是过度?

    使用css属性值在一段时间内平滑过渡,比如,鼠标悬停1s内,由白色过度到红色 一般过度发生在悬浮

  • 指定4个要素

    ==过渡属性==

    ​ transition-property:none|all|property

    • 多个属性逗号隔开
    • 可设置的过度属性
      • 颜色属性
      • 取值为数值的属性
      • 转换属性
      • 渐变属性
      • 阴影属性

    ==过度所需时间==

    ​ transition-duration: s|ms

    • 默认值为0,意味着不会有效果,所以必须设置为trasition-duration属性

    ==过度函数==

    ​ transition-timing-function:

    • 取值:
      • ease:默认值,规定慢速开始,然后变快,然后慢速结束的过度效果
      • linear: 匀速
      • ease-in:规定以慢速开始,加速效果
      • ease-out:以慢速开始,减速效果
      • ease-in-out:以慢速开始和结束,先加后减速效果

    ==过度延迟时间==

    ``transition-dapley: s|ms`

    改变元素属性后多长时间开始执行动画效果

    ==简写属性==

    • 语法:`transition:property duration timing-function delay

  • 触发过度

    通过用户的行为触发,如点击,悬浮等

动画animation

animation属性可以制作flash动画

通过关键帧控制动画每一步
使得元素从一种样式变化到另一种样式,实现复杂的动画效果

==@keyframes==

  • 作用:用于分解动画动作

    每个帧代表一个时间点,定义每个帧上的动作

animation属性用于控制动画

  • 调用由@keyframes定义的动画

    设置动画属性 时间,次数

  • 简写属性

    animation:name duration timing-function deplay iteration-count direction

  • 动画子属性

    • animation-name: 调用动画,需要与@keyframes的名字一致

    • animation-duration: s|ms 动画完成一个周期所需的时间

    • animation-timing-function: 规定动画的速度变化类型

    • animation-deplay:播放之前延迟的时间

    • animation-iteration-count: 播放次数 infinite表示无限次

    • animation-direction: 动画播放方向

      • normal 默认值 正常播放
      • alternate 轮流播放 动画会在奇数次正常播放,而在偶数次向后播放

    • animation-fill-mode:

      • forwards:动画停在最后一帧
      • backwards : 动画停止开始位置

    • both :动画开始结束都有

      • none 默认值

    • animation-play-state:

      • paused|running

        属性规定动画正在运行还是暂停,默认值为running

CSS单位的使用

宽和高

补充说明

  • 使用vw作为网站上所有内容的单位，包括字体大小和高度，将使它始终与设备的屏幕宽度成比例显示，而不管其分辨率如何。这样可以非常轻松地确保您的网站在工作站和移动设备上都能正确显示。

  • 您可以在font-size: 1vw CSS中设置body(或任何适合您项目的大小)，并且rem单位中指定的所有内容都会根据设备屏幕自动缩放，因此可以很容易地将现有项目甚至框架(例如Bootstrap)移植到此概念。

  • 相对(推荐)长度:px/pt/pc/cm/mm/in

  • 绝对长度:

    • 元素大小:vw/vh/vmax/vmin
    • 字体大小:rem/em/ex
    • ch:0的宽度
    • %
