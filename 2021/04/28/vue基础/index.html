watch属性

  • watch用于监听子组件中某一data中的属性的改变,一旦改变则会触发

  • watch中函数命名的名字必须与要监听改变的属性的名字相同

  • ```js
    // 由于dnumber与input中value值双向绑定,所以input值一变就会触发watch,newvalue是此时denumber改变的新值,也可传第二个参数旧值
    watch:{

       dnumber1(newValue,oldValue){

    // 这一步会触发dnumber2的watch监听函数执行

            this.dnumber2=newValue*100;
// 向父组件传递新值进而改变num1和number1
         this.$emit('sendfu1',newValue);
  },
  dnumber2(newValue,oldValue){
   // 这一步会触发dnumber1的watch监听函数执行
     this.dnumber1=newValue/100;
     this.$emit('sendfu2',newValue)
  }

    }
